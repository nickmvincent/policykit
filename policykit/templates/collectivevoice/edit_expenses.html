If you don't want to filter based on a given field (for instance, you want all expense types, all tags, all amounts, etc.), you can either ignore that dropdown or select "None"

Currently, Collective Voice supports four ways you can select certain expenses to filter on. You can look for matches based on:
  • the expense type. Currently, you can choose to only use CollectiveVoice for reimbursements or invoices.
  • the expense description (either look for a matching word or an exact match)
  • the expense amount. You can either match all expenses below some threshold (small expenses) or above some threshold (large expenses)
  • expense tags. Finally you can use CollectiveVoice for expenses with a certain tag.
